📋 Hướng dẫn sử dụng

Lưu ý: Để ứng dụng hoạt động đầy đủ, bạn cần:

  1. Đảm bảo file data/Question.json có sẵn
  2. Chạy trên web server (không thể mở trực tiếp file HTML)
  3. Hoặc sử dụng Live Server extension trong VS Code

🎯 Random Câu Hỏi

Tự động chọn 10 câu hỏi ngẫu nhiên từ bộ câu hỏi có sẵn, phân bố theo độ khó

📊 Hệ Thống Chấm Điểm

Chấm điểm 0-3 cho mỗi câu hỏi theo rubric chuẩn với gợi ý đánh giá

📈 Thống Kê Kết Quả

Phân tích điểm mạnh, điểm cần cải thiện và đưa ra khuyến nghị pass/fail

💾 Xuất Kết Quả

Lưu kết quả phỏng vấn dưới dạng JSON để lưu trữ và theo dõi
